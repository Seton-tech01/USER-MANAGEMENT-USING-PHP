# User Management System Using PHP
A backend-only PHP user management system built for authentication, authorization, and role-based access control.

## Features
* User registration and authentication
* Role-based access control
* Staff/user management
* Secure API endpoints
* CRUD operations
* Database integration

## Technologies Used
* PHP
* MySQL
* XAMPP
* REST API principles

## Project Structure
This project focuses on backend logic and API development without a frontend interface.

## Setup Instructions
1. Clone the repository
2. Move to XAMPP htdocs directory
3. Import the database schema
4. Configure database connection
5. Start Apache and MySQL
6. Test endpoints using Postman

## Testing
API endpoints can be tested using:
* Postman
